How to Apply CSS to HTML

To apply CSS to an HTML file, you want to use the "link" tag along with its rel and src attributes. The "rel" attribute specifies the relationship between current and linked file. The "src" attribute defines the URL to the file. Here is an example of applying CSS: 

                body{
                    color: red
                }
