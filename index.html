How to Apply CSS to HTML

To apply an CSS to an HTML file internally, you can use the "style" element and insert the CSS code in there.

To apply an external CSS to an HTML file, you want to use the "link" tag along with its rel and src attributes. The "rel" attribute specifies the relationship between current and linked file. The "src" attribute defines the URL to the file. Here is an example of applying an external CSS: 

                h4 {
                    color: red;
                    background-color: lightgrey;
                }

This text is edited by an external CSS file.
